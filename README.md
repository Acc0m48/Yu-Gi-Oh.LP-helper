# LP助手 v1.0.1

游戏王桌游辅助工具 —— 专为线下实体卡牌对战设计的 LP 计数与回合管理 App。

> 🎉 这是我第一个上传到 GitHub 的项目！

## 功能

### LP 计算器
- 双人并排 LP 显示，点击弹出数字键盘进行加减操作
- 快捷操作：支付一半 / 扣 500 / 扣 1000
- LP 变动自动记录为统一时间线中的条目，显示玩家名、变动量、前后值、回合、阶段、时间
- 支持撤销回退，删除/编辑 LP 记录后自动重算玩家生命值

### 回合管理
- 全局回合计数，点击 + 推进回合并自动切换玩家
- 阶段选择：抽卡 / 准备 / 主要1 / 战斗 / 主要2 / 结束（符合 Master Rule 2020）

### 先攻决定
- 骰子（5s 旋转动画）/ 硬币（7.5s 3D 翻转）/ 自定 三种方式
- 硬币/骰子均有重置按钮和完整动画

### 统一记录与标签
- LP 变动与回合备忘合并在一条时间线中
- 预设标签多选：玩家名、自肃、效果、召唤、攻击、连锁、伤害、特殊召唤
- 区域位置标签：点击打开可视化游戏盘（额外怪兽区、主要怪兽区、魔法·陷阱区、场地、墓地、额外卡组、卡组、手牌、除外状态）
- 所有记录可编辑回合数、阶段、内容；LP 记录可修改变动值

### 主题与工具
- 浅色 / 深色双主题一键切换
- 内置骰子和硬币小工具（含动画）

### 多对局与数据
- 侧边栏创建/切换/删除对局，自定义玩家名和初始 LP
- 数据实时 JSON 持久化，导入/导出支持跨设备迁移

